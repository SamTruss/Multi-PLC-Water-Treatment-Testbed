# Security Policy

This repository is an **intentionally vulnerable** research testbed. 
The vulnerabilities documented in the README are seeded by design 
to support evaluation of agentic and generative AI approaches to 
autonomous VAPT.

Please do not file security reports for the documented vulnerabilities.
For unintended security issues (e.g. supply-chain risks in the agent 
container), open a GitHub issue or contact the maintainer.
